- O (OCP), Principio de Abierto/Cerrado

# Comandos

- Iniciar la aplicación (Modo Dev)

```sh
npm run dev
```

- Crear una migración

```sh
npx makemigration --name nombre_migracion
```

- Ejecutar migración (Sincronizar)

```sh
npx runmigration
```
